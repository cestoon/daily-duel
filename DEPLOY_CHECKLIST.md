## 📝 CLI 上传说明

**结论**：
- ❌ CLI 无法上传此项目的云函数（目录结构问题）
- ✅ 必须使用微信开发者工具手动上传
- ✅ 云函数已经存在于云端，状态为 Active

**错误原因**：
```
EISDIR: illegal operation on a directory, read
```
CLI 工具在处理包含 `common/` 目录的云函数时有 bug

**以后的策略**：
- 开发完成后，手动上传云函数
- 不依赖 CLI 上传功能
