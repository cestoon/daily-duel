# ⚡ 超级快速指南（3分钟搞定）

## 📋 在微信开发者工具中做这5件事

### 1️⃣ 上传5个云函数（3分钟）

**找到左侧 `cloudfunctions` 目录**，右键以下5个文件夹，选择「**上传并部署：云端安装依赖**」：

```
1. settlement-getBalance            ← 右键 → 上传并部署：云端安装依赖
2. settlement-getMissedPoints       ← 右键 → 上传并部署：云端安装依赖
3. timer-weeklySettlement           ← 右键 → 上传并部署：云端安装依赖
4. scripts-diagnose                 ← 右键 → 上传并部署：云端安装依赖
5. scripts-forceNewPeriod           ← 右键 → 上传并部署：云端安装依赖
```

**等待每个上传完成**（每个约30秒，总共2-3分钟）

---

### 2️⃣ 运行修复工具（30秒）

1. 点击顶部「**云开发**」按钮
2. 进入「**云函数**」标签
3. 找到 `scripts-forceNewPeriod`
4. 点击「**测试**」
5. 直接点击「**运行测试**」（不需要输入参数）
6. 看到成功信息就OK了！

---

### 3️⃣ 测试（30秒）

1. 点击「**编译**」按钮
2. 工具 → 清缓存 → 全部清除
3. 进入「**PK对战**」页面

## 🆘 遇到问题？

### 上传失败？
- 检查网络连接
- 确认云开发已开通

### 修复工具运行失败？
- 点击云函数右侧的「日志」查看错误
- 把错误信息告诉我

### PK页面没变化？
- 再次清除缓存
- 完全关闭小程序重新打开
