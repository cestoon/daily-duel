## 🎯 优化内容

### 1. PK页面排版优化

**问题**：
- 上部分拥挤（用户信息、PK血条）
- 下部分空旷（本周累计卡片单薄）
- 视觉层次不清晰

**优化方案**：

#### 增加间距
```css
/* 用户信息区域 */
padding: 30rpx → 40rpx 30rpx
margin-bottom: 20rpx → 30rpx

/* PK血条区域 */
padding: 30rpx → 40rpx 30rpx
margin-bottom: 20rpx → 30rpx

/* 提示区域 */
padding: 8rpx 16rpx → 12rpx 20rpx
margin-bottom: 16rpx → 24rpx

/* 图例区域 */
margin-top: 16rpx → 24rpx

/* 状态提示 */
margin-top: 16rpx → 24rpx
新增: padding: 16rpx 24rpx
新增: background: rgba(255, 107, 53, 0.05)
新增: border-radius: 12rpx

/* 统计卡片 */
padding: 20rpx → 30rpx 20rpx
```

**效果**：
- ✅ 上部分不再拥挤
- ✅ 视觉层次更清晰
- ✅ 信息更易阅读
- ✅ 整体更平衡

---

### 2. Tab文字简化

**优化前**：
```
PK对战  今日打卡  条目管理  结算  设置
```

**优化后**：
```

```

**理由**：
- ✅ 减少视觉负担
- ✅ 更简洁清爽
- ✅ 图标更突出
- ✅ 易于识别

## 💡 设计原则

### 呼吸感（Breathing Space）
> 适当的留白让信息更易阅读

**应用**：
- 增加 padding 和 margin
- 避免元素过度紧凑
- 给内容"呼吸"的空间

---

### 视觉层次（Visual Hierarchy）
> 重要信息突出，次要信息后退

**应用**：
- 状态提示加背景色
- 增加间距区分区块
- 统一元素间距规范

---

### 简洁性（Simplicity）
> Less is More，去掉不必要的元素

**应用**：
- Tab文字简化
- 删除重复信息
- 保留核心内容
