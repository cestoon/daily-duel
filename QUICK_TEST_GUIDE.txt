  🎯 快速测试指南 - 切换账号方式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

前提条件：
□ 你有另一个微信号（个人号/工作号/家人号）
□ 或者能借用朋友的微信 5 分钟

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  📋 操作步骤（5分钟完成）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤 1: 记录你的邀请码
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在你的小程序中：
1. 进入「设置」页
2. 查看邀请码，例如：123456
3. 复制或记下这个邀请码

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤 2: 切换到另一个微信号
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在微信开发者工具中：

方法 A（推荐）：
┌─────────────────────────────────────────┐
│ 模拟器右上角「...」→ 切换用户            │
└─────────────────────────────────────────┘

方法 B：
┌─────────────────────────────────────────┐
│ 工具栏右上角头像 → 切换账号              │
└─────────────────────────────────────────┘

选择另一个微信号登录

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤 3: 使用第二个账号进入小程序
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 小程序会自动重新加载
2. 使用第二个微信号登录
3. 进入「设置」页

你应该看到：
┌─────────────────────────┐
│ 第二个微信号的头像和昵称  │
│                         │
│ 我的邀请码: 654321      │  ← 第二个账号的邀请码
│                         │
│ 还没有绑定搭档          │
│ [绑定搭档]              │
└─────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤 4: 绑定第一个账号
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 点击「绑定搭档」
2. 输入第一个账号的邀请码：123456
3. 点击「确定」

应该提示：「绑定成功」✅

现在看到：
┌─────────────────────────┐
│ 我的搭档                │
│ 👤 第一个账号昵称        │
│ ID: abc123              │
│ [解绑搭档]              │
└─────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤 5: 切换回第一个账号
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 再次切换用户
2. 回到第一个微信号
3. 进入「设置」页

应该看到：
┌─────────────────────────┐
│ 我的搭档                │
│ 👤 第二个账号昵称        │  ← 绑定成功！
│ ID: xyz789              │
└─────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤 6: 测试 PK 功能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 进入「PK对战」页面

应该看到：
┌─────────────────────────────────────────┐
│          PK 血条对战                    │
│                                         │
│  对手 ←[██████    ]→ 我                │
│  深蓝     橙色                          │
│                                         │
│  本周累计:                              │
│  我:  已完成 40 | 漏卡 10               │
│  TA:  已完成 30 | 漏卡 20               │
│                                         │
│  今日统计:                              │
│  我的进度: 1/2                          │
│  TA的进度: 0/2                          │
└─────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤 7: 创建打卡条目（可选）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

为了测试 PK 功能，两个账号都需要创建打卡条目：

第一个账号：
1. 进入「条目管理」
2. 添加条目：跑步、读书等
3. 完成部分打卡

第二个账号：
1. 切换到第二个账号
2. 同样添加打卡条目
3. 完成部分打卡

然后查看 PK 页面的实时对战数据！
