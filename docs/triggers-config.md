# 定时触发器配置说明

## 定时触发器Cron表达式格式

```
[秒] [分] [时] [日] [月] [周] [年]
```

## 已配置的定时任务

### 1. 每日打卡检查 (dailyCheck)
- **触发时间**: 每天 00:01
- **Cron表达式**: `0 0 1 * * * *`
- **功能**: 检查前一天的打卡情况，对未打卡的条目记录为"漏卡"
- **云函数**: `timer/dailyCheck`

### 2. 每周结算 (weeklySettlement)
- **触发时间**: 每周一 00:10
- **Cron表达式**: `0 0 10 * * 1 *`
- **功能**: 结算上周的PK结果，计算双方积分差额并创建结算记录
- **云函数**: `timer/weeklySettlement`

## 部署定时触发器

1. 在微信开发者工具中，右键点击云函数目录
2. 选择"上传并部署：云端安装依赖"
3. 定时触发器会自动部署到云开发环境

## 订阅消息配置

### 所需订阅消息模板

#### 1. 打卡提醒模板
- **模板ID**: 需要在微信公众平台申请
- **用途**: 每天定时提醒用户打卡
- **参数示例**:
  - thing1: 今日打卡时间
  - thing2: 待完成条目数
  - phrase3: 提醒语

#### 2. 结算提醒模板
- **模板ID**: 需要在微信公众平台申请
- **用途**: 每周结算完成后提醒用户确认
- **参数示例**:
  - thing1: 结算金额
  - thing2: 结算周期
  - phrase3: 确认状态

### 申请订阅消息模板

1. 登录微信公众平台
2. 进入"功能" -> "订阅消息"
3. 点击"选用"
4. 搜索相关模板或创建自定义模板
5. 填写模板内容并提交审核

### 发送订阅消息

在云函数中使用 `cloud.openapi.subscribeMessage.send()` 方法发送订阅消息：

```javascript
const res = await cloud.openapi.subscribeMessage.send({
  touser: openid,
  templateId: 'your-template-id',
  page: 'pages/checkin/checkin',
  data: {
    thing1: { value: '21:00' },
    thing2: { value: '3' },
    phrase3: { value: '记得打卡哦' }
  }
})
```

## 测试定时任务

在云开发控制台的"云函数"页面，可以：

1. 查看定时任务执行日志
2. 手动触发云函数进行测试
3. 查看定时任务的下次执行时间
