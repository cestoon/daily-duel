## 📱 步骤1: 生成体验版（5分钟）

### 1. 打开小程序后台
```
浏览器打开：https://mp.weixin.qq.com/
```

### 2. 扫码登录
```
用管理员微信扫码（应该是你的微信）
```

### 3. 进入版本管理
```
左侧菜单 → 管理 → 版本管理
```

### 4. 找到开发版本
```
页面中间会显示「开发版本」
应该能看到你刚刚上传的版本：
  • 版本号（你填写的）
  • 上传时间（刚刚）
  • 代码包大小
```

### 5. 设为体验版
```
开发版本右侧 → 点击「选为体验版」按钮
  ↓
弹出确认框 → 点击「确定」
  ↓
看到「操作成功」提示
```

### 6. 查看体验版二维码
```
页面会自动跳转到「体验版本」
  ↓
显示一个二维码
  ↓
这就是体验版的入口！
```

## 👥 步骤2: 添加体验成员（3分钟）

### 为什么需要添加？
```
体验版只有「体验成员」才能访问
默认只有管理员（你）可以访问
需要添加女朋友为体验成员
```

### 如何添加？

#### 方法1: 通过微信号添加（推荐）

```
1. 左侧菜单 → 管理 → 成员管理
   ↓
2. 顶部切换到「体验成员」标签
   ↓
3. 点击「添加成员」按钮
   ↓
4. 输入女朋友的微信号
   ↓
5. 点击「确定」
   ↓
6. 她的微信会收到通知
```

#### 方法2: 生成邀请二维码

```
1. 成员管理 → 体验成员 → 添加成员
   ↓
2. 选择「生成邀请二维码」
   ↓
3. 截图发给女朋友
   ↓
4. 她扫码即可成为体验成员
```

| **功能完整性** | 完整 | 完整 |
| **更新方式** | 重新预览 | 重新设为体验版 |

## 🐛 可能遇到的问题

### Q1: 找不到开发版本？

**原因**：可能上传没成功

**解决**：
```
1. 回到微信开发者工具
2. 再次点击「上传」
3. 填写版本号（如：1.0.0）
4. 填写项目备注
5. 点击「上传」
6. 等待上传完成
7. 刷新小程序后台页面
```

### Q2: 扫码后无法打开？

**原因**：可能没有设为体验版

**解决**：
```
1. 确认已点击「选为体验版」
2. 页面显示「体验版本」
3. 重新扫码
```

### Q3: 女朋友扫码后提示"无权限"？

**原因**：她还不是体验成员

**解决**：
```
1. 成员管理 → 体验成员
2. 添加她的微信号
3. 等她接受邀请
4. 重新扫码
```

### Q4: 打开后数据是空的？

**这是正常的！**
```
✅ 第一次使用，数据库是空的
✅ 需要先创建打卡条目
✅ 然后完成打卡
✅ 数据就会显示出来
```

### Q5: 绑定伙伴找不到对方？

**检查**：
```
1. 两人都必须是体验成员
2. 两人都必须先登录过小程序
3. 设置页面输入的绑定码必须正确
4. 绑定码在「设置」页面顶部显示
```
