# 🎨 Logo和文案优化

**版本**：v1.2.1  
**优化时间**：2026-03-25

---

## 🐛 问题

### 1. Logo图片不显示
```
问题：首页Logo显示为空白
原因：/assets/logo.png 文件路径或加载问题
表现：只显示文字，Logo区域空白
```

### 2. 文案过于强调"情侣"
```
问题：多处提到"情侣打卡PK"
影响：限制用户群体，不适合朋友、同事等关系
文案位置：
  - 首页标语："情侣打卡PK，一起变更好"
  - 底部说明："专为情侣/伙伴打造的打卡PK小程序"
  - 设置页："情侣打卡PK小程序"
```

---

## ✅ 解决方案

### 1. Logo优化

**使用emoji替代图片**：
```
图标：💪（健身肌肉）
样式：橙色渐变背景 + 圆角
尺寸：160rpx × 160rpx
```

**优点**：
- ✅ 不依赖图片文件
- ✅ 显示稳定可靠
- ✅ 符合打卡/运动主题
- ✅ 视觉效果好

---

### 2. 文案优化

**修改原则**：
- ❌ 去掉"情侣"限定词
- ✅ 使用"伙伴"通用表述
- ✅ 突出"一起"协作概念

**修改对比**：

| 位置 | 修改前 | 修改后 |
|------|--------|--------|
| 首页标语 | 情侣打卡PK，一起变更好 | **一起打卡PK，一起变更好** |
| 首页底部 | 专为情侣/伙伴打造的打卡PK小程序 | **和伙伴一起打卡PK，共同进步** |
| 设置页 | 情侣打卡PK小程序 | **伙伴打卡PK小程序** |


**WXSS**：
```css
/* 首页Logo */
.logo-placeholder {
  width: 160rpx;
  height: 160rpx;
  font-size: 100rpx;
  display: flex;
  align-items: center;

/* 弹窗Logo */
.modal-logo-placeholder {
  width: 120rpx;
  height: 120rpx;
  font-size: 80rpx;
  display: flex;
  align-items: center;
  justify-content: center;
  background: linear-gradient(135deg, #FF6B35 0%, #F7931E 100%);
  border-radius: 24rpx;
  box-shadow: 0 8rpx 24rpx rgba(255, 107, 53, 0.3);
}
```

**样式特点**：
- 渐变背景（#FF6B35 → #F7931E）
- 圆角（30rpx / 24rpx）
- 阴影（rgba(255, 107, 53, 0.3)）
- 居中对齐（flexbox）
